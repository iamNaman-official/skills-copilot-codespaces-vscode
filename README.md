## Exploring New Technology

This repository contains a demo showcasing modern JavaScript features that can be explored using GitHub Copilot in VS Code Codespaces.

### Running the Demo

```bash
node demo.js
```

### Features Demonstrated

- **Arrow Functions**: Concise function syntax
- **Template Literals**: String interpolation
- **Destructuring**: Extract values from objects and arrays
- **Spread Operator**: Merge objects and arrays
- **Async/Await**: Modern asynchronous programming
- **Array Methods**: map, filter, reduce
- **Classes**: Object-oriented programming in JavaScript
